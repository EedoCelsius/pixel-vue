# Pixel Vue

Scaffolded with [Vite](https://vite.dev) and preconfigured to use [Vue 3](https://vuejs.org/), [Pinia](https://pinia.vuejs.org/) for state management, and [Tailwind CSS](https://tailwindcss.com/) for styling.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
